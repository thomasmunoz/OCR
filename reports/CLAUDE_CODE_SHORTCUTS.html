iTerm2 Manual Configuration

If /terminal-setup doesn't work, configure manually:

1 Open iTerm2 Preferences

Cmd + , or iTerm2 → Preferences

2 Go to Profiles → Keys → General

Select your profile, then Keys tab, then General sub-tab

3 Set Option Key Behavior

Set Left Option Key to Esc+
Set Right Option Key to Esc+

4 Add Shift+Enter Mapping (Optional)

Go to Keys → Key Mappings → Click +
Keyboard Shortcut: Shift+Enter
Action: Send Escape Sequence
Esc+: [13;2u
